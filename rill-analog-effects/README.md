# rill-analog-effects

Analog circuit models тАФ operational amplifiers, tape decks, preamps.

## Key components

- **`OperationalAmplifier`** тАФ op-amp model with slew-rate limiting, bandwidth, and rail-clamping
- **`CassetteDeckModel`** тАФ tape deck emulation with tape saturation, wow & flutter, noise
- **Preamp models** тАФ configurable circuit models

## Dependencies

- `rill-core` тАФ `Node`, `Processor` trait
- `rill-core-model` тАФ WDF elements and analysis
