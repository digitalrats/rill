# rill-analog-filters

Analog-modeled filters using Wave Digital Filter (WDF) elements.

## Key components

- **`WdfMoogLadderProcessor`** — `Node` wrapper over the WDF MoogLadder 4-pole low-pass filter

## Dependencies

- `rill-core` — `Node`, `Processor` trait
- `rill-core-wdf` — WDF elements, adapters, and MoogLadder implementation
